项目背景

小栗子框架

小栗子框架曾是生态非常完善的 QQ 机器人框架之一，拥有丰富的插件与 SDK。遗憾的是，因为被犯罪团队滥用，项目逐渐停摆，生态延续面临挑战。

小栗子虚拟机

本项目以"兼容与迁移"为核心目标：通过 ws 对接的方式，将原有小栗子生态的插件能力接入更稳定且广泛使用的框架与标准（萌尘 / Dulu / OneBot V11），让历史插件资产可以被继续使用与维护。

核心特性

WebSocket 对接

以 ws 作为对接通道，降低不同框架之间的耦合，便于部署与拓展。

OneBot V11 兼容

对接 OneBot V11 标准接口，减少重复造轮子，提升生态适配能力。

SDK 持续更新

在原易语言 SDK 之外，提供 C++ 与 Python 版本 SDK，并持续补齐与更新 API。
